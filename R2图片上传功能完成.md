## 🚀 下一步：配置 Cloudflare R2

请按照 `Cloudflare_R2配置指南.md` 中的步骤配置：

### 快速步骤：

1. **创建 R2 Bucket**
   - Cloudflare Dashboard → R2 → Create bucket
   - 名称：`blog-images`（或你喜欢的名称）

2. **在 Pages 中绑定 R2**
   - Pages 项目 → Settings → Functions
   - 添加 R2 bucket binding
   - Variable name: `BLOG_R2`
   - 选择你创建的 bucket

3. **重新部署项目**
   ```bash
   git add .
   git commit -m "添加 R2 图片上传功能"
   git push
   ```

## 📝 使用方法

配置完成后：

1. 访问 `/admin` 页面
2. 登录后，创建或编辑文章
3. 在"封面图片"部分，点击"📤 上传图片"
4. 选择图片文件（支持 JPEG, PNG, WebP, GIF，最大 10MB）
5. 等待上传完成，图片 URL 会自动填充
